# RunDesk_vueJs


# Quick start

If you want to launch the project just follow these steps.

```bash
$ composer install
$ yarn install ///// $ npm install
$ yarn dev ///// $ npm run watch
$ php bin/console doctrine:migration:migrate
$ php bin/console doctrine:fixtures:load
```
